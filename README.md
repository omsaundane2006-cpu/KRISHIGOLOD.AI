# KRUSHIGOLD.AI

Fresh Flask farming project with:
- login/register
- farmer, expert, admin roles
- SQLite database
- weather API
- mandi API
- disease upload reports
- AI chat hook
- Sarvam AI multi-language translation support
- notifications
- GPS/location field
- voice upload placeholder
